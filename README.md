# FlightDelayInsight-ZSCN
ZSCN 2025-7 ✈️ 航班正常率可视化研究🧐
## 技术栈与复现指南

### 环境要求

- **Python**：3.12（支持至2028年，确保环境一致性）
- **核心库**：`pandas 2.1.3` + `openpyxl 3.1.2`（完全离线）
- **可视化**：ECharts 5离线库（`echarts.min.js`），无需网络，渲染耗时<2 s

## 核心发现

1. **早高峰效率短板**：10:00延误次峰值达180.8分钟，08:00—10:00区段平均延误55.9分钟，构成全天第一短板（空域结构+航司策略+机型错配三重叠加）。
2. **主基地航司拖累**：江西航空（CJX）正常率仅65.7%，低于样本均值9.1个百分点，其航班波密度超标（12.3班/小时）与E190支线机队占比41.3%是结构性根源。
3. **机型错配风险**：E190严重延误率高达13.8%，显著高于A320的1.4%，主因远机位分配概率73.8%与湿跑道性能不足；ARJ21在<1,200 km航段正常率达86.3%，证明机型差异关键在于"匹配度"而非技术代际。

## 项目价值

- **低成本决策**：机场运控部门无需采购付费API，利用既有脱敏数据即可生成延误热力图
- **政策启示**：地方政府航线补贴若未配套效率考核，可能诱发"补贴-延误"逆向激励
- **数据惠民**：旅客可通过本研究图表规避高风险时段与航司，实现"零成本"出行优化
